|----------|-------------|
| file bytes -> reconstruction/decode path | Encoded Parquet arrays decoded into m/z/intensity; wrong decode = wrong science |
| unsupported/corrupt input -> error path | Reader must refuse, not silently emit zeros |

| T-01-03-INFO | Information Disclosure | wrong-but-plausible spectra shipped as truth | mitigate | Named UnsupportedEncodingError surfaces the exact unsupported CV term; corrupt is a distinct class so users can act (PITFALLS 11). |

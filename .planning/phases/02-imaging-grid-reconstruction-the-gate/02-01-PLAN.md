<objective>
Build `src/reader/scanCoords.ts` — the reader-boundary coordinate + geometry extractor for the imaging grid. This is the productionized, bulk-read generalization of the existing `probeIsImaging()` boolean probe: instead of returning `true` on first hit, it returns ALL per-spectrum `{x,y}` coordinates plus the run-level grid geometry, behind a swappable `CoordSource` strategy chain (D-16).

Purpose: IMG-01 — the project's #1-risk-mitigation layer. The coordinate read must be accession-keyed, type-agnostic (Int64 OR UInt32, D-15), and join scan rows to spectra on `source_index` (not row index). Geometry must survive the vendored reader's `MSRun.fromJSON` dropping `ms_run.parameters` (Pitfall 1) by sourcing from `metadata.imaging` first.
Output: `extractCoords(reader)` and `readGridGeometry(reader)` returning plain POJOs — no Arrow/bigint leaks above `src/reader/`.
</objective>

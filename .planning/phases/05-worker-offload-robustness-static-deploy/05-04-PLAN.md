|----------|-------------|
| GitHub Actions → repo | push to main triggers build + deploy; write permission scoped to pages only |
| User browser → GitHub Pages CDN | Static assets served with correct MIME types; .nojekyll prevents Jekyll processing |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-08 | Tampering | GitHub Actions permissions | mitigate | permissions block: contents: read (not write), pages: write, id-token: write — minimal required scopes |
| T-05-09 | Information Disclosure | WASM MIME type on GitHub Pages | mitigate | .nojekyll prevents Jekyll from touching binary assets; upstream mzpeakts demo confirms Pages CDN serves .wasm correctly |
| T-05-10 | Denial of Service | codex_review.sh empty-diff produces no verdict | mitigate | Empty DIFF detection exits 1 with diagnostic before calling codex binary |
| T-05-SC | Tampering | npm/pip/cargo installs | accept | No new packages; GitHub Actions uses pinned action tags |

**Badge** — compact status / metadata pill for modes, MS levels and counts.

```jsx
<Badge tone="success" dot>Ready</Badge>
<Badge tone="info">Profile</Badge>
<Badge tone="neutral" mono>1,684 spectra</Badge>
```

Tones: `neutral`, `accent`, `info`, `success`, `warning`, `danger`. `dot` adds a status dot; `mono` renders the label in tabular monospace.

  width: number;

  <behavior>
    buildImagingGrid(coords, spectrumIndices, geometry, coordSourceStrategy) → ImagingGrid | null:
    - coords empty → return null (non-imaging valid state, D-04).
    - coordinateBase = geometry?.coordinateBase ?? 1 (read it, NEVER hard-code −1).
    - observedMaxX/Y from coords; width = geometry?.pixelCount?.x ?? observedMaxX (declared WINS, D-11); same for height; extentSource = declared present ? "declared" : "max-coord".
    - DoS cap: if width*height > 50_000_000 → console.warn + return null.
    - Allocate presenceMask = new Uint8Array(width*height).
    - For each coord i: x0 = x − coordinateBase, y0 = y − coordinateBase; bounds-check 0 ≤ x0 < width && 0 ≤ y0 < height (else skip + count out-of-range as a missing/diagnostic); key = x0*width + y0; if map has key → duplicateCount++ (keep existing, don't overwrite); else map.set(key, spectrumIndices[i]) + presenceMask[key]=1.
    - filledCount = map.size; totalCells = width*height; missingCount = total − filled.
    - discoveryDisagreement: human-readable note when declared extent present AND (declared.x < observedMaxX || declared.y < observedMaxY), per C1/C4; else null.
    - pixelSizeUm = geometry?.pixelSizeUm ?? null; geometrySource = geometry?.geometrySource ?? "derived".
